- To search for text in **Nano**, follow these steps:  

### 1. **Basic Search**  
Press:  
```
Ctrl + W
```
Then type the text you want to search for and press **Enter**.

### 2. **Search Again (Find Next Match)**  
After finding the first match, press:  
```
Ctrl + W, then Enter
```
Or simply press:  
```
Alt + W
```
to jump to the next match.
